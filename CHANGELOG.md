## [1.0.0] - 2025-12-23

- Initial version

### Features

- Support for link-type–based decoration of internal Obsidian links using aliases.
- Configurable link types with prefix and emoji replacement.
- Dynamic, link-type–aware suggestion lists for internal links in the editor.
- Frontmatter-based link type assignment via tags or `is[LinkType]` properties.
- Type-specific visual styling, including font color, background color, and underline options.
- Automatic generation of link-type–specific CSS identifiers.
